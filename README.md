# SS Final Exam Site — L7 to L12

Single-file interactive study site for the final exam material:

- L7 Advanced Large Language Models
- L8 Vision Transformer and Generative Modeling
- L9 Large Multimodal Models
- L10 Retrieval-Augmented Generation (RAG)
- L11 Federated Learning
- L12 Agentic AI Frameworks & AutoGen

## Run locally
Open `index.html` in a browser.

## Deploy to GitHub Pages
Upload the folder contents to a GitHub repository and enable Pages from the repository settings.
